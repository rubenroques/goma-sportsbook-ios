# GOMA iOS Tools

This directory contains various tools and utilities for the GOMA iOS project.

## Available Tools

### [URLsFinder](./URLsFinder)
A Python script to find hardcoded URLs in Swift source files. Helps identify and manage URLs across the codebase.

## Adding New Tools

When adding new tools:
1. Create a new directory for your tool
2. Include a README.md with usage instructions
3. Add the tool to this list
4. Follow the project's coding standards
